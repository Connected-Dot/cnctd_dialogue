# cnctd_dialogue

Custom interactive dialog/prompt library for CLI applications.

## Installation

```toml
[dependencies]
cnctd_dialogue = { git = "https://github.com/Connected-Dot/cnctd_crates.git" }
```

## Features

- Interactive prompts
- Selection menus
- Input validation
- Custom styling
